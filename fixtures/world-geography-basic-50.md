# World Geography Basic Question Bank

This file contains about 50 basic world geography multiple-choice questions in English.

Use it for:

- manual data entry
- demo content
- copy/paste testing

---

## Format

Each question includes:

- `Question`
- `A` / `B` / `C` / `D`
- `Correct`
- `Difficulty`
